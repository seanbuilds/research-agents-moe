# Research Agents MoE System

An advanced Mixture-of-Experts (MoE) research framework consisting of 8 specialized agents, each defined in its own `.md` file. Designed for high-quality, rigorous, truth-seeking research workflows in Grok, Claude, Cursor, or any custom agent platform.

## Features
- 8 modular specialist agents covering the full research pipeline
- Orchestration Layer system prompt included
- Ready-to-use Python installer script
- Fully documented quality protocols (≥3 sources, confidence scores, counterarguments, uncertainty flagging)
- Cross-platform (Windows/macOS/Linux)

## Quick Start

### 1. Install locally
```bash
git clone https://github.com/seanbuilds/research-agents-moe.git
cd research-agents-moe
python install_research_agents.py
```

### 2. Use in your AI platform
1. Copy the content of `ORCHESTRATION_LAYER_PROMPT.md` into your main agent.
2. Load individual `.md` files as custom agents.
3. Start research tasks by addressing **Research Coordinator**.

See full details in the repo files.
